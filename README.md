# KibembeNumbers


## File Descriptions

### index.html
- Main HTML structure
- Semantic markup with accessibility features
- Links to external resources (fonts, icons)
- Modular script imports

### styles/main.css
- Complete Tailwind CSS setup with custom theme
- Custom color palette using CSS variables
- Responsive design utilities
- Custom animations and components
- Accessibility-focused styling

### scripts/translator.js
- Core translation algorithms
- `ENGLISH_NUMBERS` and `KIBEMBE_ONES` dictionaries
- `englishWordsToNumber()` - Converts English words to digits
- `numberToKibembe()` - Converts digits to Kibembe
- `translateToKibembe()` - Main translation function

### scripts/app.js
- UI interaction handling
- Translation history management
- Local storage integration
- Event listeners and DOM manipulation
- Error handling and user feedback

## Development

### Local Development
1. Clone or download the project
2. Open `index.html` in a web browser
3. No build process required - works directly in the browser

### Customization
- **Styling**: Modify `styles/main.css` for visual changes
- **Translation Logic**: Update `scripts/translator.js` for language changes
- **UI Behavior**: Modify `scripts/app.js` for interaction changes

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
